# Piano Universe

A React + Vite interactive demo with particle-based number animation and piano-style sound synthesis.

## About

This project renders a full-screen animated particle canvas inside a React app. Each tap/click triggers:
- a glowing particle interaction,
- a piano-like synthesized note,
- a countdown from `10` down to `0`, then cycles back.

The app is meant as a creative concept demo for interactive visuals and sound in the browser.

## Tech stack

- React 18
- Vite
- JavaScript / JSX
- Web Audio API

## Files

- Main app component: `src/App.jsx`
- Entry point: `src/main.jsx`
- Styles: `src/index.css`

## Run locally

```bash
npm install
npm run dev
```

Then open: **`http://localhost:5173/`**

The app will load with the Piano Universe interactive demo.
